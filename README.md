## Overview

* 16 8-bit Registers
* 4k Bytes of Memory
* 16-bit Index Register 
* 16-bit Program Counter 
* 16-level Stack
* 8-bit Stack Pointer
* 8-bit Delay Timer
* 8-bit Sound Timer
* 16 Input Keys
* 64x32 Monochrome Display Memory
